# Web3 Dev Ranking

A sleek, professional ranking list of top blockchain developers and organizations focused on Solana/Metaplex, Neo3, and Web3 Security ecosystems.

## Features

- **Curated Rankings** — Top 6 blockchain developers and organizations with detailed profiles
- **Filter by Ecosystem** — Filter by Solana, Neo3, Web3 Security, or Infrastructure tags
- **GitHub Integration** — Direct links to each organization's/developer's GitHub profile
- **Responsive Design** — Works on desktop, tablet, and mobile
- **Live Animations** — Smooth fade-in and slide-up transitions
- **Dark Theme** — Professional dark UI with ambient glow effects

## Tech Stack

- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript

## Running

Simply open `index.html` in a web browser:

```bash
# Using Python's built-in server
cd web3-devs
python3 -m http.server 8080

# Then visit http://localhost:8080
```

Or open the file directly:

```bash
open index.html  # macOS
xdg-open index.html  # Linux
```
